# 🐾 Petshop Banner Design

Banner desenvolvido no Figma com foco em identidade visual moderna, experiência do usuário e comunicação clara para um petshop.

## 🎯 Objetivo

Criar um design atrativo e funcional que transmita confiança, cuidado e profissionalismo, incentivando a interação do usuário.

## ✨ Destaques do Projeto

* 🎨 Paleta de cores harmoniosa e estratégica
* 🧠 Hierarquia visual bem definida
* 📱 Layout pensado para boa experiência do usuário
* 🐶 Comunicação visual voltada para o público pet

## 🛠️ Ferramentas

* Figma
